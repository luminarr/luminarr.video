Discover

Browse curated movie lists from The Movie Database and add titles to your library in a single click. Discover is the fastest way to build out your collection without leaving Luminarr.

Navigation

Browse Categories

The top of the Discover page offers category tabs powered by TMDb's discovery API. Switch between lists to find movies by relevance, popularity, or release timing.

Trending

Movies with the highest TMDb activity over the past week. Updated daily.

Popular

The most-viewed movies on TMDb right now. A broad selection across genres.

Top Rated

Highest average vote scores on TMDb, filtered to movies with significant vote counts.

Upcoming

Movies with theatrical release dates in the next 30 days. Add them now and Luminarr will search on release.

By Genre

Select any TMDb genre from the genre filter to narrow results to a specific category: Action, Comedy, Drama, Horror, Science Fiction, and more. Genre filtering works across all category tabs.

UI

Movie Cards

Each movie in the Discover grid is displayed as a card with the information and actions you need to make a quick decision.

Card element Description
Poster TMDb poster image. Click to open the full movie detail page on TMDb.
Title The movie's primary title as listed on TMDb.
Year Theatrical release year.
Rating TMDb average vote score out of 10, shown with vote count.
Action buttons Add to Library, or view the detail page. Context-sensitive based on library status (see below).
Adding Movies

Quick Add

Click Add on any card to open the Quick Add panel inline. You can configure the library details before confirming without navigating away.

  1. 1

    Select library — choose the root folder path where the movie will be stored.

  2. 2

    Select quality profile — pick the profile that defines acceptable quality and upgrade behavior for this movie.

  3. 3

    Toggle monitor — monitored movies are automatically searched. Add unmonitored to track without searching immediately.

  4. 4

    Confirm — the movie is added to the database and an immediate search is triggered if monitored.

Defaults from the last add are remembered per session. The library and quality profile fields pre-populate with your most recent selections, making rapid bulk-adding fast.

Status Indicators

In Library and Excluded Indicators

Cards are visually marked to prevent duplicate adds and surface content you have explicitly excluded.

In Library

A green overlay and checkmark indicate the movie is already in your Luminarr library. Clicking the card navigates to the movie's detail page instead of showing the add panel.

Excluded

A red overlay marks movies you have explicitly excluded from Luminarr. Excluded movies will not be added by RSS sync or automatic search. You can un-exclude from the movie detail page.
