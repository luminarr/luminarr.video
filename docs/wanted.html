Wanted

The Wanted page is your action list: movies that Luminarr is actively trying to obtain or improve. Three focused tabs let you triage missing content, quality gaps, and pending upgrades.

Missing Content

Missing tab

The Missing tab shows every monitored movie in your library that has no file on disk. These are movies Luminarr knows about and wants to find, but hasn't yet.

Column Description
Movie Title and year. Clicking opens the movie detail page.
Quality Profile The quality profile assigned to this movie, which governs what releases are acceptable.
Added When the movie was added to Luminarr.
Last search When Luminarr last ran an indexer search for this movie. Empty if never searched.
Actions Manual search trigger, unmonitor button.

Search All Missing

Click Search All Missing to queue a manual indexer search for every movie in the list simultaneously. Results arrive as grabs via the normal queue flow.

Search All can generate many simultaneous indexer queries. If you have rate-limited indexers, searches may be throttled or partially fail. Check the Activity log for grab results.

Quality Gaps

Cutoff Unmet tab

The Cutoff Unmet tab shows movies that have a file, but the file's quality is below the cutoff defined in the movie's quality profile.

The cutoff is the minimum acceptable quality. Anything below it is always considered worth upgrading — Luminarr will grab a better release whenever one appears in RSS or manual search, as long as upgrades are enabled on the profile.

Current quality below cutoff

The file on disk doesn't meet the minimum quality you set. Luminarr will actively search for a better version on each RSS sync.

Upgrade pending

A better release was found but is queued or waiting for a download slot. The movie stays in this list until the upgrade completes.

Each row shows the current quality alongside the profile cutoff, so you can see at a glance how far the current file falls short. Click Search to trigger an immediate manual search for that movie.

Quality Improvements

Upgrades tab

The Upgrades tab shows movies where a better quality is available or expected, grouped by upgrade tier. Unlike Cutoff Unmet, these movies already meet the cutoff — but their profile allows upgrading further and a better release exists.

Upgrade tiers

Movies are grouped by the quality jump they represent, making it easy to prioritize which upgrades are most impactful:

Tier Example Priority
SD → HD 480p WebRip → 1080p Bluray High — major visual improvement
720p → 1080p 720p WebDL → 1080p Bluray Medium — noticeable resolution gain
1080p → 4K 1080p Bluray → 2160p Remux Low — significant storage cost
Source upgrade 1080p WebDL → 1080p Bluray Low — same resolution, better encode

Upgrade until quality on the quality profile controls when Luminarr stops searching. Once a movie reaches the "upgrade until" quality level, it is removed from this list and no further upgrades are attempted.
