Calendar

A monthly calendar view of release dates for movies in your library. See at a glance which movies were or will be released each day, color-coded by their download status.

Navigation

Navigation

The calendar opens on the current month. Use the navigation controls at the top to move between months.

Previous / Next month

Arrow buttons on either side of the month heading step one month at a time in either direction.

Today button

Jumps back to the current month from anywhere. Today's date is highlighted with a distinct border in the grid.

The calendar uses theatrical release dates from TMDb, not the date the file was downloaded or imported. This lets you see historical release dates alongside upcoming titles.

Color Coding

Legend

Movie chips on the calendar are colored to reflect their download and monitoring status, so you can see the state of your library at a monthly glance.

Color Status Meaning
Green Downloaded The movie has a file in the library. Quality upgrade may still be pending.
Orange Monitored, no file The movie is monitored and Luminarr is searching for a release, but no file exists yet.
Gray Unmonitored The movie is in the library but not monitored. Luminarr will not search for or upgrade it.
Grid

Calendar Grid

The calendar grid shows one cell per day. Days with movies in your library display one or more movie chips inside the cell.

Day cells

  • Each cell shows the day number in the top-left corner.
  • Today's cell has a highlighted border to orient you quickly.
  • Days in adjacent months (before the 1st or after the last day) are shown at reduced opacity for context without visual noise.
  • Cells with no movies in your library for that date are left empty.

Movie chips

Each movie on a given release date is shown as a small chip inside the day cell. The chip shows a thumbnail poster (where available) alongside the movie title, truncated if needed.

Overflow: +X more

When more movies fall on a single day than can fit in the cell without overflow, the cell shows the visible chips plus a +X more indicator. Click the indicator to expand a popover listing all movies on that date.

Release date density varies: wide-release weeks (major holidays, summer) may have many movies in a short span, while off-season months may be sparse. The calendar handles both gracefully.

Interaction

Navigating to movies

Every movie chip on the calendar is a link. Clicking a chip navigates directly to that movie's detail page in the library.

From the detail page, you can:

  • View and manage the movie's file(s)
  • Trigger a manual search
  • Change the quality profile
  • Toggle monitoring
  • See the full event history for that movie

Use the browser back button to return to the calendar at your previous month position.

Keyboard navigation: Use the left and right arrow keys while focused on the calendar header to step between months — no mouse required.
